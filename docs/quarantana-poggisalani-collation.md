# Quarantana 1840 ↔ Poggi Salani — textual collation

Divergences between the **Quarantana 1840** print (FICLIT facsimile; our verified
diplomatic transcription) and the **Poggi Salani** critical edition (Edizione
Nazionale), for the reviewed text of *I Promessi Sposi* (chapters 1–38 + intro).

## Systematic differences (editorial conventions — not listed individually)

| convention | occurrences | nature |
|---|---:|---|
| accent | 1,997 | Quarantana grave (`perchè`) → Poggi Salani acute (`perché`) |
| quote-order | 92 | mark inside vs outside the closing guillemet (`.»` ↔ `».`) |
| punctuation | 71 | sentence-mark choice (`;` ↔ `:`) |
| case | 43 | capitalization |
| apostrophe | 29 | imperative `va` → `va'`, elisions — a modernization |
| footnote marker | 24 | Quarantana prints an inline `*` / reference number |

These are regular conventions of the critical edition (grave→acute accents, etc.).
