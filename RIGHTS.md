# Rights statement

This dataset combines components with different origins. The licence on the
record (**CC BY-NC 4.0**) applies to the project-created components; the
table below states the basis for each component.

| Component | Origin | Rights basis |
|---|---|---|
| Manzoni's text of *I promessi sposi* (1840) | Alessandro Manzoni (d. 1873) | **Public domain** |
| Critical reading text (Poggi Salani 2013) | Centro Nazionale Studi Manzoniani, ed. Teresa Poggi Salani | Encoded and redistributed within the LeggoManzoni / Manzoni Online project |
| TEI encoding of the text | LeggoManzoni / Manzoni Online project encoders | CC BY-NC 4.0 |
| Facsimile layer and alignment coordinates (`facsimile.xml`, `@facs` links) | This project (automated generation + manual review) | CC BY-NC 4.0 |
| Illustration location fields in `illustrations_metadata.csv` (`x,y,w,h`, `match_*`, `iiif_region_url`) | This project | CC BY-NC 4.0 |
| Illustration descriptive metadata in `illustrations_metadata.csv` (titles, subjects, creators, …) | FICLIT Digital Library catalogue records | Reproduced from the FICLIT catalogue with attribution |
| Page images | FICLIT Digital Library, Università di Bologna | **Not included in this dataset.** Only coordinates and IIIF URLs are distributed; the images are served by FICLIT under its own terms (<https://dlrc.ficlit.unibo.it>) |

No page images, thumbnails, or other image files are included in this
dataset.
